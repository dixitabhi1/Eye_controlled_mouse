# Eye Controlled Mouse

Patent-backed Assistive Technology Project | Built for Accessibility

This project implements a **real-time eye-tracking mouse controller** using **OpenCV, MediaPipe, and PyAutoGUI**. It allows users to move and click the cursor using **only eye movements and blinks**, providing a completely **hands-free way to interact with a computer**.

The system is designed primarily to help **individuals with physical or motor disabilities**, enabling them to use computers without a traditional mouse or keyboard.

# Features

- Real-time **eye tracking using MediaPipe facial landmarks**
- **Gaze-based cursor movement**
- **Blink detection for mouse click**
- Works with a **standard webcam**
- **No additional hardware required**
- Lightweight and easy to run

---

- Python

- PyAutoGUI

# Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Eye_controlled_mouse.git
cd Eye_controlled_mouse
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```
